В современном розничной торговле косметикой вопрос контроля сроков годности тестеров и пробников стоит особенно остро. Данный проект решает реальную проблему автоматизации этого процесса.
Эта программа работает как автоматический помощник по контролю сроков годности косметических тестеров. Когда сотрудник добавляет новый тестер, он вводит его название, бренд и указывает срок годности — либо конкретной датой, либо количеством месяцев, в течение которых продукт годен после вскрытия. Программа запоминает эту информацию и автоматически заводит таймер, который отсчитывает секунды до момента, когда тестер нужно будет выбросить.

Проект автоматизации контроля сроков годности тестеров является актуальным по следующим причинам:

Законодательное требование — предоставление просроченных тестеров запрещено

Реальные штрафы — крупные сети уже платят до 150 000 рублей

Угроза здоровью — просроченные тестеры опасны бактериями

Экономическая эффективность

Представьте, что у вас есть косметический магазин с 50 тестерами (пробниками). Каждый тестер нужно выбросить через определенное время (например, через 6 месяцев после открытия).

Ваш код делает 3 простые вещи:

Запоминает - когда открыли каждый тестер

Считает - сколько времени осталось

Напоминает - когда пора выкинуть
